openstack虚机搭建VPN Server：ERROR Code 806

发表于 2018-09-29 | 分类于 openstack ， VPN | 评论数：

问题描述： openstack环境搭建VPN server，访问报806错误
现象：openstack环境创建Windows server，其中搭建VPN server，访问报806错误
查看安全组，规则全部允许通过

问题原因：iptable没有加载gre和pptp模块
解决方法：$ modprobe nf_conntrack_proto_gre
$ modprobe ip_nat_pptp
$ modprobe ip_conntrack_pptp
